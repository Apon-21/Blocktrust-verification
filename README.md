# BlockTrust

Fully deployed Web3 application for academic certificate verification.

## Live Deployment
- Frontend: Vercel (free)
- Blockchain: Ethereum Sepolia Testnet

## Features
- No MetaMask required for verification
- Read-only blockchain access
- Tamper-proof credentials
- CV and scholarship ready project

## How to Deploy
1. Upload this repo to GitHub
2. Import frontend folder to Vercel
3. Done
